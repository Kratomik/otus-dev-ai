# 📋 Шаг 5. Техническое задание: EcoTrack v1.0 (MVP)

## 1. Описание проекта
**Тип:** Мобильное (iOS/Android) + Web (адаптивное)  
**Цель:** Измерение и снижение личного углеродного следа через рекомендации и геймификацию  
**Платформы:** iOS 13+, Android 8+, Web (Chrome/Safari/Firefox)

## 2. Целевая аудитория
- Основная: 18-35 лет, РФ/СНГ → глобально, эко-осознанные, средний+ доход
- Вторичная: семьи, корпоративные пользователи (B2B v2.0)

## 3. Функциональные требования
### 3.1 Калькулятор (F-001)
- Ввод: транспорт, питание, энергия, покупки
- Расчёт: методология IPCC 2023, коэффициенты для региона
- Выход: тонны CO₂/год, pie-chart, бенчмарк, автосохранение
- Ошибки: валидация, офлайн-кэш, graceful degradation

### 3.2 Рекомендации (F-002)
- Генерация: 3-5 советов, расчёт экономии, сложность внедрения
- Логика: ранжирование по воздействию, учёт профиля и геолокации
- Ошибки: fallback на базовые, кэш, обработка таймаутов

### 3.3 Трекер прогресса (F-003)
- Компоненты: цели, бейджи, уровни, очки, графики, push
- Поведение: real-time обновление, анимации, экспорт PDF, шеринг
- Ошибки: автосохранение, восстановление, защита от дублей

## 4. Нефункциональные требования
- **Производительность:** загрузка ≤2с (3G), отклик ≤300мс, 60 FPS
- **Доступность:** WCAG 2.1 AA, VoiceOver/TalkBack, контраст ≥4.5:1
- **Безопасность:** TLS 1.3, JWT, bcrypt, CSP, quarterly audits
- **Масштабируемость:** модульный монолит → микросервисы, Redis, CDN
- **Локализация:** ru, en (v1.0); kk, uk (v1.5)
- **Надёжность:** Uptime ≥99.5%, daily backups, RTO≤4ч, RPO≤1ч

## 5. Технические требования
- **Mobile:** React Native 0.73+, TypeScript, Redux Toolkit, React Navigation
- **Web:** React 18, Vite, Tailwind CSS, Recharts
- **Backend:** Node.js 20 LTS, Fastify, PostgreSQL 15, Prisma ORM, Redis 7
- **Auth/Infra:** Firebase Auth, JWT, Vercel (web), Render (api), GitHub Actions CI/CD
- **Тесты:** Jest (≥80%), Cypress/Detox (E2E), k6 (load)

## 6. Дизайн-требования
- **Стиль:** Концепция 2 (Яркий/Игровой) + опция «Минималистичный режим»
- **Цвета:** Primary #00E676, #2979FF | Light #F5F9F7 | Dark #0D1B2A
- **Типографика:** Inter (основной), Roboto Mono (данные), масштабируемые размеры
- **Анимации:** 200-300мс, cubic-bezier(0.4,0,0.2,1), поддержка prefers-reduced-motion
- **Адаптив:** Mobile-first, breakpoints XS→XL, touch targets ≥44px
